tsdl and raylib libraries need to be installed.

To install TSDL:

For Linux: Run the command [sudo apt update] and then run the command [sudo apt install libsdl2-dev]. Lastly run 
the command [opam install tsdl].

For macOS: Run the command [brew install sdl2] and then run the command [opam install tsdl]. If you receive an error that there is a package missing try to run [brew install pkg-config sdl2] and then rerun [opam install tsdl].

For Windows: Run the command [winget install Git.ocaml.opam] or install the Windows Subsystem for Linux 2 to install opam. Then run the command [opam install tsdl].

To install raylib:

For Linux: Run the command [apt-get install librarylib-dec]

For macOS: Run the command [brew install raylib] and then run the command [opam install raylib].

For Windows: Download the pre-built libraries from the official website
